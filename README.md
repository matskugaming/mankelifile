# mankelifile
Simple PHP File Server
### Requirements:
- Internet connection
- Linux *(distro that has systemd)*
- PHP 7.0 or newer
  * Recommended: PHP 7.4+ or PHP 8.1+

No Apache or Nginx required

## Install
Open terminal and run:
```bash
curl -fsSL https://raw.githubusercontent.com/matskugaming/mankelifile/refs/heads/master/mankelifile-setup.sh > mankelifile-setup.sh
bash mankelifile-setup.sh
```
